# GDAgent
### The Professional AI Development Environment for Godot

GDAgent transforms the Godot Editor into a fully integrated AI coding environment. Unlike simple chat plugins, GDAgent provides a persistent, workspace-aware terminal engineered for AI-assisted development inside the editor.

## Installation

### Requirements
- **Godot Engine**: 4.2 or later
- **Operating System**: Linux, macOS, or Windows

### Install with GDAgent Manager
1. Download and run the **GDAgent Manager** for your platform.
2. Select your Godot project and click **Install GDAgent**.
3. Open Godot and enable the plugin in **Project Settings > Plugins**.
